# Changelog

All notable changes to the Squad Feedback Client will be documented in this file.

## [1.0.0] - 2025-10-20

### Added
- Initial release of Squad Feedback Client
- Support for submitting single feedback items
- Support for batch feedback submission
- Configurable timeout and retry options
- Full TypeScript support with type definitions
- Support for different feedback sources (TYPEFORM, SLACK, MANUAL, UNKNOWN)
- Support for multiple environments (development, staging, production)
- Comprehensive error handling
- Example code and documentation
- Getting started guide

### Features
- Lightweight implementation using native `fetch`
- No heavy dependencies
- Rate limiting support for batch submissions
- Automatic timeout handling
- Detailed error information in responses

### Documentation
- Complete README with API reference
- Getting started guide
- Integration examples (Express.js, Slack, Typeform)
- TypeScript usage examples
